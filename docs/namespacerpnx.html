◆ get_path()

std::vector< std::filesystem::path > rpnx::get_path ( )

Gets the directories in the path in a thread safe manner by locking rpnx::environment_mutex.

This function works by looking at the PATH variable (when on POSIX systems) or the equivalent on other operating systems, and splitting it via the OS's preferred path separator (usually : or ;).

Example

auto path = rpnx::get_path();
for (std::filesystem::path const & x : path)
{
std::cout << "Path Value=" << x << std::endl;
}
