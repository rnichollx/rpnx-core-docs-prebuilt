rpnx::experimental::async_ip6_tcp_autoacceptor Class Reference

#include <network.hpp>

Detailed Description

This is an asynchronous I/O generator for a async_ip6_tcp_acceptor. While this object exists, accept events are generated on the socket.

Note that the lifetime of such events is bound to the async service, or, if transferred, to the handling agents.
