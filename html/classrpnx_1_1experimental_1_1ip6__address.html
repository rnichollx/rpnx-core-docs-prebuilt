◆ native()

IMPLEMENTATION_DEFINED rpnx::experimental::ip6_address::native ( ) const

Returns the native IPv6 address representation. The format of struct in_addr differs depending on your operating system. Note: On Windows, this returns IN6_ADDR, which contains a union. On compilers that don't support type-punning in C++, the active union field is the .Byte representation.
